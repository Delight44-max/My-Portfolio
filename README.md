# 🚀 Personal Portfolio – Fullstack Engineer

A modern, responsive personal portfolio built with **Next.js, TypeScript, and Tailwind CSS**.  
It showcases my projects, skills, and experience as a Fullstack Engineer based in Lagos, Nigeria.

Designed with a strong focus on performance, clean UI, responsiveness, and smooth animations.

---

## ✨ Features

- ⚡ Fast and optimized with Next.js
- 📱 Fully responsive (mobile, tablet, desktop)
- 🎨 Modern UI with custom design system
- 🧠 Smooth scroll reveal animations
- 🧩 Modular component-based architecture
- 📂 Projects section with featured work
- 📬 Contact section for easy communication
- 🚀 Deployed on Vercel

---

## 🛠️ Tech Stack

- **Frontend:** Next.js, React, TypeScript
- **Styling:** CSS Modules / Tailwind CSS (adjust if needed)
- **Animations:** Custom CSS animations + Intersection Observer
- **Deployment:** Vercel

---

## 📁 Project Structure
/components → Reusable UI components
/lib → Data and utilities (projects, etc.)
/app → Next.js app router pages
/public → Static assets (images, icons)
/styles → Global styles
